# 我的餐廳清單

**功能**
---
- 查看所有餐廳
- 瀏覽餐廳詳細資訊
- 透過餐廳名稱或餐廳類別搜尋餐廳
- 連結餐廳的地址到 Google 地圖

**開始使用**
---
- 請先確認有安裝 node.js 與 npm

- 將專案 clone 到本地

- 在本地開啟之後，透過終端機進入資料夾，輸入：`npm install`

- 安裝完畢後，繼續輸入：`npm run start`

- 若看見此行訊息則代表順利運行，打開瀏覽器進入到以下網址 : `Listening on http://localhost:3000`

- 若欲暫停使用: `同時按下ctrl + c`
  
**開發工具**
---
- Node.js 10.15.0
- Express 4.16.4
- Express-Handlebars 3.0.0
- Express-session 1.17.3
- connect-flash 0.1.1
- Bootstrap 5.2.1
- Font-awesome 5.8.1
